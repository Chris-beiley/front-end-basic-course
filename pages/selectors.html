Selector types

  • a (element) - chooses all links on the page
  • #introductionParagraph (#id) - chooses element with unique name
  • .introduction(.class) - applies the style to elements with this class
  • .timeline.date1 (.class1.class2) - applies the style that described in the rules for each class
  • .timeline + .timeline (class1 + class2) - the style is applied to adjacent elements
  • a[href^="https"] - apllies to link begins with 'https' atribute
